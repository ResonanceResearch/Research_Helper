## Deploy
1) Static front-end (GitHub Pages).
2) Cloudflare Worker:
```bash
npm i -g wrangler
wrangler login
wrangler secret put OPENAI_API_KEY
# set your KV id in wrangler.toml
wrangler deploy
```
3) In `index.html` set:
```html
<script>window.API_BASE="https://faculty-research-mentor-api.YOURNAME.workers.dev";</script>
```
